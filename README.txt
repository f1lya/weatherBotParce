==Python Weather Bot==

Телеграм-бот для вывода прогноза погоды на ближайшие 30 дней
в городе Ярославль

==Установка==

Для начала работы с ботом нужно запустить файл
weatherBotParce.py, предварительно убедившись,
что все пакеты из файла requirements.txt установлены 
на сервер/компьютер. Затем вставить токен вашего телеграм-
бота в переменную TOKEN

==Старт==

Бот выводит прогноз погоды на заданный день. Для этого 
нужно отправить ему нужную дату в формате дд месяц
Пример: 3 августа
